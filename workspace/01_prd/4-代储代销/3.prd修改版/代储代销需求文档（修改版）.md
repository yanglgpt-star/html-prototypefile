## 1. 项目背景与目标

### 1.1 需求简介

代储代销是供应链金融平台的核心业务模块，为供应商提供"先入库、后销售"的供应链金融服务。供应商将货物存入平台指定仓库，平台支付履约保证金，货物通过平台挂牌销售，销售完成后进行清分结算。该模式帮助供应商提前回笼资金，同时平台通过货物监管降低风险。

### 1.2 业务诉求

| 序号 | 诉求 | 说明 |
|------|------|------|
| 1 | 供应商资金提前回笼 | 通过支付履约保证金，帮助供应商在销售前获得资金支持 |
| 2 | 货物全程监管 | 从入库到出库的全流程货物监管，降低平台风险 |
| 3 | 平台挂牌销售 | 支持定向销售和不定向销售两种模式 |
| 4 | 自动清分结算 | 销售完成后自动按比例扣回保证金、服务费、仓储费 |
| 5 | 多方协同 | 供应商、采购商、平台、仓储方、金融方多方业务协同 |

### 1.3 目标用户与菜单结构

| 角色 | 菜单结构 | 说明 |
|------|---------|------|
| 供应商（无仓平台） | **醒市无仓**<br>├─ 供应链服务<br>│  ├─ 代储代销<br>│  └─ 仓储回购通（即将上线）<br>├─ 服务管理<br>└─ 我的订单<br>   └─ 代储代销 | 提供货物并申请代储代销服务的企业 |

| 金融助手 | **金融助手**<br>├─ 代储代销审核<br>├─ 电子仓单管理<br>├─ 保证金拨付<br>├─ 销售活动发布<br>├─ 销售订单管理<br>├─ 出库指令管理<br>├─ 自动分账<br>├─ 票据中心<br>└─ 提货凭证管理 | 管理代储代销全流程业务 |

### 2.2 交易模式说明

**参与方**：
- **供应商**：提供货物，签署协议，获得保证金
- **醒市无仓**：平台方，提供代储代销服务
- **采购商**：购买代储代销货物
- **仓库**：提供仓储服务，管理货物入库出库

**核心流程**：
1. **准入**：核验货物，勾选《代储代销服务协议》，办理中登网登记
2. **入仓**：实物核验，仓库完成入库后生成入库清单，金融助手根据入库清单生成电子仓单并锁货
3. **支付保证金**：按T+0实时评估货值，支付不超过70%的保证金至供应商监管账户
4. **发布销售活动**：金融助手发布销售活动，同步至市场助手进行挂牌销售
5. **下单支付**：采购商在市场助手下单，全款支付至第三方支付通道（分账系统）
6. **出库指令**：金融助手识别货款到账后，发布出库/发货指令至仓库
7. **确认收货**：买方确认收货（或物流签收），触发分账指令
8. **自动结算**：系统按预设比例扣回：保证金、服务费、仓储费；余款自动结算至供应商银行账户，各方在线下载/打印结算清单及发票申请

**开票规则**：
- 卖方全额开票给买方
- 平台开具服务费+仓储费给卖方

## 3. 详细功能说明

### 3.1 注册与准入模块

#### 3.1.1 账号注册（复用现有功能）

- **说明**：此功能复用现有平台的账号注册模块，无需新增开发
- **位置**：醒市无仓 → 账号注册（复用现有功能）
- **目标**：完成供应商账号注册，开通平安账户

**前置条件**：
- 供应商需先完成平台账号注册和实名认证
- 开通平安钱包或其他金融钱包账户
- 注册完成后，方可进入代储代销服务申请流程

#### 3.1.2 服务选择

- **位置**：醒市无仓 → 供应链服务 → 代储代销
- **目标**：选择"代储代销"供应链服务协议

**功能说明**：
1. **页面布局**：
   - 顶部导航栏：黄色渐变背景（#ffd700 → #ffcc00）
   - Logo区域：醒市無仓 + 狮子头图标
   - 供应链服务卡片：显示"供应链服务"标题和"去看看>"链接
   - 服务管理按钮：点击进入服务订单列表
   - 用户信息：显示当前登录供应商名称

2. **服务类型选择**：
   - 代储代销：先入库后销售，提前回笼资金，最高可获得货值70%的保证金
   - 仓储回购通：平台代采货物并仓储，降低库存压力（即将上线）

3. **协议确认**：
   - 勾选《代储代销服务协议》复选框
   - 点击协议名称可查看协议全文
   - 必须同意协议后才能进行下一步

**交互说明**：
- 未勾选协议时，"下一步"按钮置灰不可点击
- 点击协议名称，弹窗展示《代储代销服务协议》全文
- 阅读并勾选协议后，"下一步"按钮亮起可点击
- 点击供应链服务卡片可直接进入服务选择页
- 点击服务管理按钮进入我的代储代销列表页

#### 3.1.3 我的代储代销（服务管理）

- **位置**：醒市无仓 → 服务管理 / 我的订单 → 代储代销
- **目标**：供应商查看自己提交的代储代销申请记录及状态

**功能说明**：

1. **申请列表**：
   - 展示供应商提交的所有代储代销申请记录
   - 支持按申请编号、申请状态筛选
   - 支持查询、重置操作

- 保留两位小数
- 千分位分隔符
- 前缀"¥"符号
- 示例：¥100,000.00

### 4.3 日期时间格式

- 日期：YYYY-MM-DD，如：2026-04-01
- 日期时间：YYYY-MM-DD HH:mm:ss，如：2026-04-01 14:30:00

### 4.4 计算规则

| 计算项 | 公式 | 说明 |
|--------|------|------|

| 仓储费 | 存储天数 × 日仓储费率 × 货物数量 | 按天计算 |
| 供应商实收 | 订单金额 - 扣回保证金 - 服务费 - 仓储费 | 供应商最终收到金额 |

### 6.1 术语表

| 术语 | 说明 |
|------|------|
| 代储代销 | 供应商将货物存入平台仓库，平台支付保证金，货物通过平台销售的供应链金融服务模式 |
| 仓储回购通 | 平台代采货物并仓储，帮助供应商降低库存压力的服务（即将上线） |
| 履约保证金 | 平台根据货物评估价值向供应商支付的资金，用于提前回笼供应商资金 |
| 定向销售 | 供应商指定特定采购商购买货物的销售模式 |
| 不定向销售 | 货物对所有平台采购商开放的销售模式 |
| 电子仓单 | 货物入库后生成的电子凭证，代表货物所有权和质押权 |
| 分账系统 | 第三方支付通道，用于管理货款的分账结算 |
| 电子提货码 | 采购商支付货款后生成的提货凭证，用于仓库提货 |
| T+0评估 | 当日实时货物价值评估 |

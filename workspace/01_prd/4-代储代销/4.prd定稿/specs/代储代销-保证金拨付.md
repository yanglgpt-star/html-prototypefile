# Spec - 保证金拨付

**对应需求点**：3.3 保证金拨付  
**交付对象**：前端 + 服务端  
**依赖**：平安钱包支付接口（待确认）

---

## Goal

金融助手运营能够对已入库的代储订单进行保证金拨付，将履约保证金支付到供应商的平安钱包账户。

**用户**：金融助手运营人员  
**操作**：查看拨付列表 → 点击拨付 → 确认拨付信息 → 完成支付  
**达成**：保证金到账，仓单状态变为"已质押"

---

## Scope

### 包含
- 保证金拨付列表页（搜索、列表展示）
- 拨付确认弹窗（拨付信息、收款账户、提醒）
- 拨付详情弹窗查看
- 拨付操作：确认拨付

### 不包含
- 平安钱包支付的具体接口调用逻辑（仅UI流程）
- 支付失败后的退款流程
- 供应商钱包账户管理
- 批量拨付功能

---

## I/O

### 列表页接口

**搜索条件**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| paymentNo | string | 否 | 拨付编号 |
| supplierName | string | 否 | 供应商名称 |
| paymentStatus | enum | 否 | 拨付状态：PENDING/PAID |

**列表字段**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| paymentNo | string | 拨付编号，如 BF20250909001 |
| orderNo | string | 代储订单编码 |
| supplierName | string | 供应商名称 |
| actualInboundValue | decimal | 实际入库货值（元） |
| marginRatio | decimal | 保证金比例 |
| paymentAmount | decimal | 拨付金额（元） |
| paymentTime | datetime | 拨付时间，待拨付显示"-" |
| paymentStatus | enum | 拨付状态 |
| canPay | boolean | 是否显示"拨付"按钮 |

### 拨付确认弹窗

**获取拨付信息**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| paymentNo | string | 拨付编号 |
| supplierName | string | 供应商名称 |
| paymentAmount | decimal | 拨付金额 |
| receiverAccount | object | 收款账户信息 |
| └─ accountName | string | 账户名：能良科技 |
| └─ accountNo | string | 账号：**** **** **** 6688 |
| └─ accountType | string | 账户类型：平安数字钱包 |
| └─ bankLogo | string | 银行Logo（平安银行） |

**确认拨付提交**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| paymentNo | string | 是 | 拨付编号 |
| confirmPay | boolean | 是 | 确认拨付标识 |

**异常情况**
- 仓单状态不是"普通存单"：提示"该订单仓单状态异常，无法拨付"
- 供应商钱包账户异常：提示"供应商收款账户异常，请联系供应商确认"
- 支付超时：提示"支付处理中，请稍后查看结果"

### 详情弹窗接口

复用列表字段，增加：
| 字段名 | 类型 | 说明 |
|-------|------|------|
| warehouseReceiptNo | string | 关联仓单编号 |
| inboundListNo | string | 关联入库单编号 |
| receiverAccountDetail | string | 完整收款账户信息 |
| operationRecords | array | 操作记录 |
| └─ action | string | 操作动作 |
| └─ desc | string | 操作描述 |
| └─ time | datetime | 操作时间 |

## Constraints

### 技术约束
- 拨付金额显示为红色大字（#cf1322，18px，font-weight: 600），突出显示
- 收款账户信息区使用蓝色背景（#e6f7ff），蓝色边框（#91d5ff）
- 提醒区使用黄色背景（#fffbe6），黄色边框（#ffe58f）
- 拨付信息区使用绿色背景（#f6ffed），绿色边框（#b7eb8f）
- 平安银行Logo使用渐变背景（#ff6b35 到 #f7931e）
- 弹窗宽度：560px

### 业务约束
- 拨付条件：货物已完成入库、电子仓单已生成（状态=普通存单）、审核定价已完成
- 拨付金额 = 实际入库货值 × 保证金比例
- 拨付操作不可撤销，请谨慎操作

### 状态流转
```
待拨付 → 确认拨付 → 支付成功 → 已拨付
              → 支付失败 → 待拨付（可重试）
```

# Spec - 出库指令管理

**对应需求点**：3.6 出库指令管理  
**交付对象**：前端 + 服务端  
**依赖**：WMS系统（指令发布）、市场助手系统（支付状态）

---

## Goal

金融助手运营能够管理销售订单的出库指令，在采购商支付货款后发布出库/发货指令至仓库。

**用户**：金融助手运营人员  
**操作**：查看出库指令列表 → 搜索/筛选 → 发布指令/查看详情  
**达成**：确保货款到账后及时安排仓库发货

---

## Scope

### 包含
- 出库指令管理列表页（搜索、列表展示）
- 出库指令状态管理（待发布/已发布/已出库）
- 出库指令发布操作
- 出库指令详情查看

### 不包含
- 支付状态实时监测逻辑（由外部系统处理）
- 仓库实际出库操作（WMS系统处理）
- 物流跟踪信息展示（外部系统）
- 指令撤回/修改功能（一旦发布不可撤销）

---

## I/O

### 列表页接口

**搜索条件**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| instructionNo | string | 否 | 指令编号 |
| saleOrderNo | string | 否 | 销售订单编号 |
| instructionStatus | enum | 否 | 指令状态：PENDING/ISSUED/OUTBOUND |

**列表字段**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| instructionNo | string | 指令编号，如 ZL20250909001 |
| saleOrderNo | string | 销售订单编号 |
| buyerName | string | 采购商名称 |
| pickupQuantity | int | 提货数量 |
| pickupCode | string | 提货码 |
| instructionStatus | enum | 指令状态 |
| issueTime | datetime | 发布时间，待发布显示"-" |
| operation | string | 操作：发布指令/详情 |

### 出库指令状态定义

| 状态码 | 状态名 | 触发条件 | UI表现 |
|-------|-------|---------|--------|
| PENDING | 待发布 | 销售订单已支付，等待发布指令 | 黄色标签 |
| ISSUED | 已发布 | 指令已发布至仓库 | 绿色标签 |
| OUTBOUND | 已出库 | 仓库已完成出库 | 蓝色标签 |

### 发布指令接口

**请求参数**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| instructionNo | string | 是 | 指令编号 |
| confirmIssue | boolean | 是 | 确认发布标识 |

**异常情况**
- 销售订单未支付：提示"该订单货款未到账，无法发布指令"
- 指令状态非待发布：提示"该指令已发布，请勿重复操作"
- 仓库系统异常：提示"仓库系统异常，请稍后重试"

### 详情弹窗接口

**出库指令详情**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| instructionNo | string | 指令编号 |
| saleOrderNo | string | 销售订单编号 |
| buyerName | string | 采购商名称 |
| instructionStatus | enum | 指令状态 |
| pickupQuantity | int | 提货数量 |
| pickupCode | string | 提货码 |
| issueTime | datetime | 发布时间 |
| receiverName | string | 收件人 |
| receiverPhone | string | 联系电话 |
| receiverAddress | string | 收件地址 |

## Constraints

### 技术约束
- 指令状态标签颜色：待发布（黄色）、已发布（绿色）、已出库（蓝色）
- 提货码显示为蓝色高亮
- 待发布状态的发布时间显示"-"

### 业务约束
- 触发条件：系统实时监测采购商支付状态，货款到账后自动生成待发布指令
- 发布操作不可撤销，请谨慎操作
- 指令发布后，仓库WMS系统接收指令并安排出库

### 状态流转
```
待发布 → 发布指令 → 已发布 → 仓库出库 → 已出库
```

## Acceptance Criteria

### 列表页
- ✅ 打开出库指令管理页面，展示出库指令列表
- ✅ 列表展示 8 列：指令编号、销售订单编号、采购商、提货数量、提货码、指令状态、发布时间、操作
- ✅ 待发布状态显示黄色标签
- ✅ 已发布状态显示绿色标签
- ✅ 已出库状态显示蓝色标签
- ✅ 待发布状态行显示"发布指令"和"详情"按钮
- ✅ 已发布/已出库状态行只显示"详情"按钮
- ✅ 发布时间列，待发布状态显示"-"，已发布/已出库显示具体时间
- ✅ 输入指令编号搜索，列表展示匹配结果
- ✅ 选择指令状态筛选，列表展示匹配结果

### 发布指令
- ✅ 点击"发布指令"按钮，弹出确认对话框
- ✅ 确认后提示"出库指令发布成功"
- ✅ 发布后该记录状态变为"已发布"，发布时间更新为当前时间

### 详情弹窗
- ✅ 点击"详情"按钮，弹出出库指令详情弹窗
- ✅ 展示基本信息：指令编号、销售订单编号、采购商、指令状态、提货数量、提货码、发布时间
- ✅ 展示提货信息：收件人、联系电话、收件地址

---

## Non-Goals

- 不实现支付状态监测逻辑（由外部系统触发）
- 不实现仓库出库操作（WMS系统处理）
- 不实现指令撤回功能（一旦发布不可撤销）
- 不实现物流跟踪信息展示

# Spec - 提货凭证管理

**对应需求点**：3.9 提货凭证管理  
**交付对象**：前端  
**依赖**：WMS系统（提货码核销）

---

## Goal

金融助手运营能够查看和管理电子提货码，作为仓库提货凭证。

**用户**：金融助手运营人员  
**操作**：查看提货码列表 → 搜索/筛选 → 查看详情  
**达成**：掌握提货码的生成和使用情况

---

## Scope

### 包含
- 提货凭证管理列表页（搜索、列表展示）
- 提货码详情查看

### 不包含
- 提货码生成逻辑（销售订单支付后自动生成）
- 提货码核销（仓库WMS系统扫码核销）
- 提货码作废/重新生成功能

---

## I/O

### 列表页接口

**搜索条件**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| pickupCode | string | 否 | 提货码 |
| saleOrderNo | string | 否 | 销售订单编号 |
| useStatus | enum | 否 | 使用状态：UNUSED/USED |

**列表字段**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| pickupCode | string | 提货码，如 TH20250909ABCD |
| saleOrderNo | string | 销售订单编号 |
| buyerName | string | 采购商名称 |
| pickupQuantity | int | 提货数量 |
| generateTime | datetime | 生成时间 |
| expireTime | datetime | 有效期至 |
| useStatus | enum | 使用状态：未使用/已使用 |
| operation | string | 操作：查看 |

---

## Constraints

### 技术约束
- 使用状态标签：未使用（绿色背景 #f6ffed，绿色文字 #52c41a）、已使用（蓝色背景 #e6f7ff，蓝色文字 #1890ff）
- 提货码有效期默认 7 天（从生成时间算起）
- 提货码使用等宽字体，蓝色高亮显示
- 页面底部显示说明文案

### 业务约束
- 提货码由销售订单支付成功后自动生成
- 提货码使用状态由WMS系统扫码核销后更新
- 本页面仅做查询展示，不提供修改操作

---

## Acceptance Criteria

### 列表页
- ✅ 打开提货凭证管理页面，展示提货码列表
- ✅ 列表展示 8 列：提货码、销售订单编号、采购商、提货数量、生成时间、有效期至、状态、操作
- ✅ 未使用状态显示绿色标签（背景 #f6ffed，文字 #52c41a）
- ✅ 已使用状态显示蓝色标签（背景 #e6f7ff，文字 #1890ff）
- ✅ 输入提货码搜索，列表展示匹配结果
- ✅ 选择使用状态筛选，列表展示匹配结果

### 说明区域
- ✅ 页面底部显示说明文案："采购商支付货款后，系统自动生成电子提货码，作为仓库提货凭证"

# Spec - 清结算中心

**对应需求点**：3.7 清结算中心（原自动分账）  
**交付对象**：前端 + 服务端  
**依赖**：清结算系统（分账数据来源）

---

## Goal

金融助手运营能够查看销售完成后的分账明细，了解每笔销售的资金清分情况。

**用户**：金融助手运营人员  
**操作**：查看分账列表 → 搜索/筛选 → 查看分账详情  
**达成**：掌握每笔销售的费用扣减和供应商实收情况

---

## Scope

### 包含
- 清结算中心列表页（搜索、列表展示）
- 分账明细展示（订单金额、扣回保证金、服务费、仓储费、供应商实收）
- 清结算详情弹窗（分账明细、结算信息、分账记录）

### 不包含
- 分账触发逻辑（由清结算系统处理）
- 分账计算逻辑（由清结算系统处理）
- 实际资金划拨操作（由清结算系统处理）
- 票据开具功能（票据中心处理）

---

## I/O

### 列表页接口

**搜索条件**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| settlementNo | string | 否 | 分账编号 |
| saleOrderNo | string | 否 | 销售订单编号 |
| settlementStatus | enum | 否 | 分账状态：PENDING/PROCESSING/COMPLETED |

**列表字段**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| settlementNo | string | 分账编号，如 FZ20250909001 |
| saleOrderNo | string | 销售订单编号 |
| orderAmount | decimal | 订单金额（元） |
| marginDeducted | decimal | 扣回保证金（元） |
| serviceFee | decimal | 服务费（元） |
| storageFee | decimal | 仓储费（元） |
| supplierActual | decimal | 供应商实收（元） |
| settlementStatus | enum | 分账状态 |
| operation | string | 操作：详情 |

### 清结算详情弹窗

**获取分账详情**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| settlementNo | string | 分账编号 |
| saleOrderNo | string | 销售订单编号 |
| settlementStatus | enum | 分账状态 |
| detailList | array | 分账明细列表 |
| └─ itemName | string | 费用项目 |
| └─ amount | decimal | 金额 |
| └─ calcMethod | string | 计算方式 |
| └─ description | string | 说明 |
| settlementInfo | object | 结算信息 |
| └─ supplierName | string | 供应商 |
| └─ receiverAccount | string | 收款账户 |
| └─ triggerTime | datetime | 触发时间 |
| └─ completeTime | datetime | 完成时间 |
| settlementRecords | array | 分账记录 |
| └─ action | string | 动作名称 |
| └─ desc | string | 动作描述 |
| └─ time | datetime | 时间 |

### 分账明细计算规则

| 费用项目 | 计算公式 | 示例 |
|---------|---------|------|
| 销售总额 | 销售单价 × 数量 | ¥34,995.00 = ¥3,499.50 × 10台 |
| 评估总价 | 评估单价 × 数量 | ¥6,999.00 = ¥699.90 × 10台 |
| 扣回保证金 | 评估总价 × 保证金比例 | ¥24,496.50 = ¥6,999.00 × 70% |
| 服务费 | 评估总价 × 服务费率 | ¥349.95 = ¥6,999.00 × 1%（5‱=0.5%？待确认）|
| 仓储费 | 元/台/天 × 存储天数 × 数量 | ¥300.00 = 10元/天/台 × 50天 × 10台 |
| 供应商实收 | 销售总额 - 扣回保证金 - 服务费 - 仓储费 | ¥9,848.55 |

**【待确认】** 服务费率单位：PRD写"5‱"（万分之5=0.05%），但示例计算"评估总价 × 1%"，存在矛盾。

## Constraints

### 技术约束
- 分账明细表格需要展示计算方式列
- 供应商实收行使用绿色背景（#f6ffed）突出显示，绿色文字（#52c41a），字体加粗
- 分账记录按时间顺序展示（时间轴样式）
- 列表金额颜色规范：订单金额蓝色（#1890ff）、扣回保证金/服务费/仓储费红色（#cf1322）、供应商实收绿色（#52c41a）

### 业务约束
- 分账触发条件：采购商确认收货或物流自动签收
- 本页面仅展示分账结果，不处理分账触发和资金划拨
- 分账状态由清结算系统推送更新

- ✅ 扣回保证金、服务费、仓储费显示为红色（#cf1322）
- ✅ 供应商实收显示为绿色（#52c41a）

- ✅ 分账基本信息区域显示：分账编号、销售订单编号、分账状态
- ✅ 分账明细表格展示 6 行：销售总额、评估总价、扣回保证金、服务费、仓储费、供应商实收
- ✅ 供应商实收行背景色为绿色（#f6ffed）
- ✅ 结算信息区域显示：供应商、收款账户、触发时间、完成时间
- ✅ 分账记录区域按时间顺序展示：触发分账 → 扣回保证金 → 扣除服务费 → 结算给供应商
- ✅ 每条分账记录显示动作名称、描述、时间

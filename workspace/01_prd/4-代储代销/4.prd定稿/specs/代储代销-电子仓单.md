# Spec - 电子仓单管理

**对应需求点**：3.2 电子仓单管理  
**交付对象**：前端 + 服务端  
**依赖**：WMS系统入库数据推送

---

## Goal

金融助手运营能够查看和管理电子仓单，跟踪仓单从创建到核销的全生命周期状态。

**用户**：金融助手运营人员  
**操作**：查看仓单列表 → 搜索/筛选 → 查看仓单详情  
**达成**：实时掌握货物仓储状态和质押情况

---

## Scope

### 包含
- 电子仓单列表页（搜索、列表展示）
- 5种仓单状态展示及标签颜色
- 仓单详情弹窗查看
- 入库清单编号链接（可点击跳转/查看）

### 不包含
- 仓单状态修改操作（状态由系统自动流转）
- 仓单线下纸质管理
- WMS系统入库操作（外部系统）
- 仓单质押/解押的审批流程（由保证金拨付触发）

---

## I/O

### 列表页接口

**搜索条件**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| warehouseReceiptNo | string | 否 | 仓单编号 |
| supplierName | string | 否 | 供应商名称 |
| status | enum | 否 | 仓单状态：PENDING/ACTIVE/COLLATERALIZED/RELEASED/CLOSED |

**列表字段**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| warehouseReceiptNo | string | 仓单编号，如 CD20250909001 |
| orderNo | string | 代储订单编码 |
| inboundListNo | string | 入库清单编号，蓝色链接可点击 |
| supplierName | string | 供应商名称 |
| goodsInfo | string | 货物信息 |
| goodsQuantity | int | 货物数量 |
| inboundDate | date | 入库日期 |
| evalValue | decimal | 评估价值（元） |
| status | enum | 仓单状态 |
| statusLabel | string | 状态显示文本 |

### 仓单状态定义

| 状态码 | 状态名 | 触发条件 | UI表现 |
|-------|-------|---------|--------|
| PENDING | 待生效 | 供应商提交申请，未入库 | 灰色背景 #f5f5f5，虚线边框 |
| ACTIVE | 普通存单 | WMS核实验收完成 | 绿色背景 #f6ffed，绿色文字 #52c41a |
| COLLATERALIZED | 已质押 | 保证金拨付完成 | 红色背景 #fff2f0，红色文字 #cf1322，加粗 |
| RELEASED | 部分释放 | 部分货款支付，对应比例库存释放 | 橙色背景 #fff7e6，橙色文字 #d46b08 |
| CLOSED | 已核销 | 全部货物出库完毕 | 灰色背景 #f5f5f5，灰色文字 #8c8c8c，实线边框 |

## Constraints

### 技术约束
- 状态标签颜色必须严格按照 PRD 规范实现
- 入库清单编号显示为蓝色链接，点击可查看入库详情

### 业务约束
- 仓单状态由系统自动流转，不提供人工修改入口
- 已质押状态的仓单禁止出库和调拨（需系统控制，本页面仅展示）

---

## Acceptance Criteria

### 列表页
- ✅ 打开电子仓单管理页面，默认展示全部仓单列表
- ✅ 仓单状态列根据状态显示对应颜色的标签
- ✅ 待生效状态显示灰色虚线边框标签（背景 #f5f5f5，边框 dashed #d9d9d9）
- ✅ 普通存单状态显示绿色标签（背景 #f6ffed，文字 #52c41a）
- ✅ 已质押状态显示红色加粗标签（背景 #fff2f0，文字 #cf1322，font-weight: 600）
- ✅ 部分释放状态显示橙色标签（背景 #fff7e6，文字 #d46b08）
- ✅ 已核销状态显示灰色实线边框标签（背景 #f5f5f5，边框 solid #d9d9d9）
- ✅ 入库清单编号显示为蓝色链接，点击弹出入库信息详情弹窗
- ✅ 点击"详情"按钮，弹出仓单详情弹窗

### 详情弹窗
- ✅ 仓单详情弹窗展示仓单基本信息、货物明细、评估汇总
- ✅ 货物明细表格展示：品牌、商品、SKU、SKU编码、数量、申请单价、评估单价、评估总价
- ✅ 仓单基本信息区域展示：提货数量、未提货数量
- ✅ 评估汇总区域展示：货物总评估价、保证金比例、保证金金额
- ✅ 点击入库清单编号蓝色链接，弹出入库信息详情弹窗
- ✅ 入库信息详情弹窗展示：入库基本信息（编号、订单、供应商、日期、仓库、类型、状态、操作员）
- ✅ 入库货物明细表格展示：序号、品牌、商品名称、SKU、SKU编码、入库数量、单位、批次号、存放位置

# Spec - 票据中心

**对应需求点**：3.8 票据中心  
**交付对象**：前端  
**依赖**：发票系统接口（待确认）

---

## Goal

金融助手运营能够查看和管理代储代销业务相关的发票票据。

**用户**：金融助手运营人员  
**操作**：查看票据列表 → 搜索/筛选  
**达成**：掌握发票开具和下载情况

---

## Scope

### 包含
- 票据中心列表页（搜索、列表展示）
- 开具发票按钮（UI占位）
- 下载结算清单按钮（UI占位）
- 发票下载按钮（UI占位）

### 不包含
- 发票开具功能（需财务系统对接）
- 发票下载功能（需文件存储对接）
- 结算清单生成和下载
- 发票作废/红冲功能

---

## I/O

### 列表页接口

**搜索条件**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| invoiceNo | string | 否 | 票据编号 |
| invoiceType | enum | 否 | 票据类型：SALES/SERVICE/STORAGE |
| invoiceStatus | enum | 否 | 发票状态：PENDING/ISSUED |

**列表字段**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| invoiceNo | string | 票据编号，如 PJ20250909001 |
| invoiceType | enum | 票据类型：销售发票/服务费发票/仓储费发票 |
| issuer | string | 开票方 |
| receiver | string | 收票方 |
| invoiceAmount | decimal | 发票金额（元） |
| relatedOrder | string | 关联订单编号 |
| invoiceStatus | enum | 发票状态 |
| operation | string | 操作：下载 |

---

## Constraints

### 技术约束
- 开具发票按钮、下载结算清单按钮、下载操作按钮均为UI占位
- 点击后提示功能开发中

### 业务约束
- 本页面为展示页面，核心功能（开具、下载）需后续开发

---

## Acceptance Criteria

- ✅ 打开票据中心页面，展示票据列表
- ✅ 列表展示 8 列：票据编号、票据类型、开票方、收票方、发票金额、关联订单、发票状态、操作
- ✅ 发票状态标签：待开具（黄色 #fffbe6/#faad14）、已开具（绿色 #f6ffed/#52c41a）
- ✅ 点击"开具发票"按钮，提示"开具发票功能开发中"
- ✅ 点击"下载结算清单"按钮，提示"下载结算清单功能开发中"
- ✅ 点击"下载"按钮，提示"发票下载功能开发中"
